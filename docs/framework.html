Libraries

  • p5.js

    p5.js is a javascript client-side library for creating graphic and interactive experiences, based on the core principles of Processing.

    With interactivity and multiplataform support in mind, p5.js was chosen to be used to create the main canvas of the application.

  • priority-queue.js

    In order to keep a priority queue of the collision events in the gas, a heap data structure was used, the following js package provides the functionality required: js-priority-queue.
